# Optimizio Performance - Demo Scenario

## Demo Project

## Sample Scores
- Performance: 74
- SEO: 81

## Sample Issues
- Large hero image causing slow loading
- Missing meta description on key pages
- Some buttons lack accessible labels
- Third-party analytics script adds overhead

## AI Recommendations
- Convert hero images to WebP/AVIF
- Add metadata and improve heading structure
- Improve keyboard focus state and button labeling
- Reduce third-party script impact
