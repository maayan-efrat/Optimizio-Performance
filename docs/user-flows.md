# Optimizio Performance - User Flows

## New User Flow
1. Landing page
2. Register or continue as demo
3. Create project
4. Add website URL
5. Start scan
6. View AI report and recommendations

## Scan Flow
1. User starts scan
2. System creates background job
3. Website is analyzed
4. Performance checks run
5. SEO and accessibility checks run
6. AI processing generates recommendations
7. Results are stored and shown in the dashboard

## Competitor Flow
1. Open competitor analysis
2. Enter target website URL
3. Add up to 3 competitor URLs
4. Run comparison analysis
5. View ranking, strengths, weaknesses, and insights
